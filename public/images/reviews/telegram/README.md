# Отзывы из Telegram-чатов (изображения)

Сюда кладите файлы картинок для блока **«Отзывы из Telegram-чатов»** на странице `/ru/reviews/`.

## Имена файлов

Имя файла должно совпадать с записью в коллекции `src/content/reviews/`:

| Файл на диске | Запись в контенте |
|---------------|-------------------|
| `telegram-01.png` (или `.jpg`, `.webp`) | `telegram-01.md` |
| `telegram-02.*` | `telegram-02.md` |
| … | … |

В frontmatter каждого `telegram-NN.md` поле **`screenshot`** указывает путь от корня сайта, например:

```yaml
screenshot: /images/reviews/telegram/telegram-01.png
```

Если вы сохранили картинку как `.jpg` или `.webp`, **откройте соответствующий `.md`** и поправьте расширение в `screenshot:`.

Пока в проекте есть только `telegram-01.png` и `telegram-02.png` — несколько записей в контенте могут указывать на один и тот же файл (временные дубликаты для сетки).

## Где лежит папка в проекте

```
public/images/reviews/telegram/
```

Всё из `public/` отдаётся как статика по URL `/images/...`.

## После добавления файлов

Пересборка не обязательна для картинок из `public/` — достаточно обновить страницу. Если меняли только `screenshot:` в `.md`, выполните `npm run build` или перезапустите `npm run dev`.
