# Правила для страниц продуктов

> Шаблон: source-DOM.html строки 9940–10080
> Данные продуктов: source-DOM.html строки 7913–9930 (16 продуктов)
> Страница: src/pages/ru/products/[slug].astro
> Данные: src/content/products/*.md (Content Collection)
> Прочитай DOM-Design-System.md

## Архитектура
- Каждый продукт — .md файл в Content Collection
- ОДИН шаблон страницы для всех 16 продуктов
- 8 компонентов-блоков (переиспользуемые)
- Добавить продукт = создать .md файл. Всё.

## SEO (для каждого продукта):
- title: "{название} | Программы DOM"
- description: из shortDesc
- Schema.org: Product + BreadcrumbList
- Хлебные крошки: Главная → Карта пути → Направление → Продукт

## Правила
1. ВСЕ через props. Никакого хардкода.
2. Блоки условные: если нет reviews — не показывать. Если нет faq — не показывать.
3. НЕ ТРОГАЙ Footer, Navbar.
