# Neuro Landing Page

Modern single-page landing: HTML + Tailwind CSS (CDN). Responsive, Flexbox + Grid, dark purple gradient, glassmorphism, red → pink → purple accents.

## Run

Open `index.html` in a browser, or serve the folder:

```bash
npx serve .
# or
python3 -m http.server 8080
```

## Assets

- **Phones**: Replace the placeholder app/Buddha blocks with real device mockup images if you have them.
- **Brain**: Replace the central “Glowing brain illustration” placeholder with your SVG or image.

## Tech

- HTML5
- Tailwind CSS (CDN)
- Responsive (mobile-first)
- Flexbox + CSS Grid
